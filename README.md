Feedstock for [rivet](https://rivet.hepforge.org) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Rivet - the particle-physics MC analysis toolkit

## Architectures

| Architecture | Latest published |
|--------------|------------------|
| linux-amd64 (`linux-64`) | ✅ `4.1.0` |
| linux-arm64 (`linux-aarch64`) | ✅ `4.1.0` |

_As of the last feedstock render; the badges above are live._


## Install

```bash
conda install -c hep-forge -c conda-forge rivet
```
